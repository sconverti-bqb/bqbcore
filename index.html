Modo de uso

BQB Core (o como lo quieran llamar) es un framework de HTML y CSS que contiene los elementos de los sitios de Buquebus y Buquebus Turismo más comunes para que sean reutilizados. En este momento, tiene prioridad por sobre Bootstrap 3 y lo podría reemplazar definitivamente, aunque el modelo de grillas está basado en Bootstrap 4 pensadas para mobile-first y hechas con Flexbox.

Ventajas

  • Mantener en un solo CSS todos los componentes personalizados de la marca, basados en la guía de diseño.
  • Realizar cambios generales y que se reflejen en todo el sitio.
  • Mayor prolijidad y consistencia a través de todo el sitio.
  • Mayor agilidad para detectar errores en implementaciones del lado de desarrollo.

Assets

Al igual que Bootstrap, el framework utiliza un solo CSS, el cual siempre su ultima versión va a estar subida en una URL fija.

<link rel="stylesheet" href="https://www.buquebus.com/static/bqbcore/src/css/bqbcore.css">

En el caso de se use este framework más Bootstrap, este debe especificarse último por cuestiones de herencia de estilos.

BQB Core funciona optimamente con la fuente Open Sans en toda sus variaciones. Esta fuente se debe definir en el HEAD de esta manera:

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,600,700,800">

Y en el CSS de esta:

font-family: 'Open Sans', sans-serif;

Grillas

Basadas en las nuevas grillas de Bootstrap 4, este nuevo sistema de grillas tiene una pequeña curva de aprendizaje y varía al metodo de uso de Bootstrap 3, pero a la larga terminarían siendo mejores por utilizar Flexbox y su metodología mobile-first.

  • Los contenedores padres son .bqb-container o .bqb-container-fluid
  • La clase para las filas es .bqb-row
  • Las columnas se arman como en Bootstrap pero con el prefijo "bqb". Ejemplo: .bqb-col-md-12
  • Junto con el prefijo "bqb", la otra diferencia es que es una grilla mobile-first, por lo que cada media query tiene un min-width en vez de un max-width.

En el caso de quererse usar las grillas de Bootstrap 3, esta no le afectaría en lo absoluto por llevar el prefijo proprietario de Buquebus.
